
# Qunara Design System

## Visual feeling

Qunara should feel like:

- An interactive museum
- A premium educational book
- A contemplative art exhibition
- A scientific documentary

## Colors

- Midnight blue background
- Warm ivory text
- Muted gold highlights
- Deep green accents
- Subtle violet for quantum subjects

## Images

Do not use:

- Generic glowing Buddha
- Golden Buddha statue as decoration
- Floating monks
- Chakra imagery
- Mystical hands holding energy
- Neon atoms
- Cheap AI art

Use:

- Ancient manuscripts
- Galaxies
- Stars
- Water under magnification
- Ink-wash mountains
- Lotus geometry
- Wave patterns
- Scientific instruments
- Abstract particles
- Quiet human silhouettes

The Buddha should appear only when historically necessary.

## Layout

Use:

- Large editorial illustrations
- Strong visual hierarchy
- Comfortable reading width
- Generous spacing
- Clear section transitions
- Responsive mobile layout
- Subtle animation
- Reduced-motion support
